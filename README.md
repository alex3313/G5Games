# G5Games
Repositorio de página WEB creada con Django para la venta de juegos de diferentes tipos, con CRUD para insertar diferentes juegos, también contiene una API para entregar información sobre diferentes ofertas y una base de datos configurada para OracleCL. Facilmente adaptable a diferentes servidores.
